# MVP – ProManage (V1)

/* English description */



## Included

* **Authentication** (JWT)
* **Roles:** ADMIN / USER
* **Client Management** (CRUD)
* **Appointment Management** (CRUD)
* **PDF Invoice Generation**
* **Simple Dashboard** (Global stats)
* **English & French** language support

## Excluded (V2+)

* **Online Payments**
* **Multi-company** (Multi-tenancy)
* **Advanced Permissions**
* **Subscriptions**
* **Full Multi-language** (Additional languages)

---

### A few notes on the translation:

* **CRUD:** This remains the same as it is a universal technical acronym.
* **Global stats:** Used "Global" to match your "stats globales."
* **Multi-tenancy:** If "Multi-entreprises" refers to a software architecture where different companies have isolated data, "Multi-tenancy" is the professional term often used in English.
